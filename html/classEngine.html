Detailed Description

Engine class handles the rendering, writing rendered pixels to framebuffer and storing framebuffer to disk.

framebuffer is array which stores all rendered pixels in heap. When framebuffer is stored on disk, netpbm format is used see https://en.wikipedia.org/wiki/Netpbm.
