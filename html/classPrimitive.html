Detailed Description

Primitive is abstract class which provides interface for all primitives in the world. (such as spheres, triangles and polygons)

All primitives have hit() member function

hit member function solves hit equation for each primitive and ray.

note that hit equation is different for each primitive.

Parameters

cmaxis maximum acceptable value for hit equation's solution.
