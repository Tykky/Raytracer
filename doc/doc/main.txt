 @section features Features

<b> Features added during the course </b>
    - Core functionality: Camera, Ray, Vector3D, @ref Raytracer "sampling" and 
      @ref Raytracer::rayTrace "recursive ray scattering".
    - Geometry: Sphere
    - Materials: Lambertian, Metal and Dielectric

    - Simple command line interface
    - Parallelization with OpenMp

<b> Features added after the course </b>
    - OpenGL GUI
    - MSVC support 

@section build_src Building the project
    If your system has all the required depedencies you should be
    able to clone and build the project with:   
    @code
        git clone https://github.com/Tykky/Raytracer --recursive
        cd raytracer
        mkdir build
        cd build
        cmake ..
        cmake --build . --config release
    @endcode

    Note that OpenGL capable graphics card is required to run and build the project with GUI.
    If you don't have one, make sure to build the project with CMake option -DBUILD_GUI=OFF.
    Also note that multithreading does not work if your compiler doesn't support OpenMP.

    After building the executables can be found in

    @code
        raytracer/build/bin
    @endcode

    The project can be built without GUI by using

    @code
        cmake -DBUILD_GUI=OFF ..
        cmake --build . --config release
    @endcode

    and without tests by using

    @code
        cmake -DBUILD_TEST=OFF ..
        cmake --build . --config release
    @endcode

    @section build_docs Building the documentation
